## Usage

```bash
$ npm i -g firebase-tools
$ npm i -g @skeet-framework/cli
```

```bash
$ Taxfycreate <project-name>
```

```bash
$ cd <project-name>
$ Taxfys
```

Open a new terminal and run:

```bash
$ Taxfylogin
$ export ACCESS_TOKEN=<your-token>
```
